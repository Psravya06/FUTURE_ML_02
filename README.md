# Support Ticket Classification System

## Overview

This project uses Machine Learning and NLP techniques to classify customer support tickets into different categories and assign priority levels.

## Tools Used

* Python
* Pandas
* Scikit-learn
* NLTK

## Features

* Ticket classification
* Text preprocessing
* Priority assignment
* Model evaluation

## Dataset

The dataset contains support tickets categorized into Account, Billing, Technical, Delivery, and General issues.

## Output

The system predicts the ticket category and assigns a priority level (High, Medium, or Low).

## Conclusion

This project demonstrates how Machine Learning can help automate customer support operations and improve response efficiency.
